About This Server

This web server dynamically renders real crop yield data from the Global Gridded Crop Yield (GHDY) Dataset using Node.js and SQLite. Each page you visit is built on the fly, with data queried directly from our local database and visualized using Chart.js.

Connection to Sustainability

Crop yield measures how much food we can produce from a given amount of land. Tracking changes in yields helps scientists and policymakers understand how efficiently we’re using global farmland. Increases in yield over time often reflect improvements in technology, crop genetics, and resource management — all key aspects of sustainable agriculture.

However, sustainability isn’t only about higher yields; it’s about producing more with less impact. This dashboard invites you to explore that balance: are rising yields a sign of sustainable progress or of environmental stress?
